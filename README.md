# Open-WebUI-Kontext-Traffic-Light
A precise context window monitor (context traffic light) that reads the hardware limit directly from the Open WebUI SQLite database and displays the exact token usage from the Ollama metadata. This is visualized using traffic light colors. Token values ​​are also output. Attenion: value "num_ctx" must be changed!
